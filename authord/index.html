Complete Documentation Integrity

Designed for writers who need engineering-grade tools, not just a spellchecker.

The Sentinel

An always-on monitor that highlights "Doc Drift" in the code editor. It ignores whitespace but flags semantic changes (like new parameters).

Agentic Harmonize

Don't start from zero. The "Scribe" agent reads the code diff + your style guide and proposes a Markdown patch for you to review.

Tri-State Diff View

The ultimate clarity. See Code Change, Local Draft, and Remote Snapshot side-by-side in a single pane of glass.

Conflict-Safe Sync

We fetch read-only snapshots first. If the remote version has drifted, we block the publish until you resolve the conflict.

Governance as Code

Define rules in rules.yaml (e.g., "Public APIs must have examples"). The AI enforces them before drafting text.

Versioned Citations

Chat with your docs. Authord cites specific Confluence versions (e.g., "v12"), so you know if an answer is based on stale data.

Style Guide as Code

Stop reviewing PRs for tone. Define your writing standards in a config file. Authord enforces them automatically during the drafting phase.

  • Enforce required sections (e.g., Error Handling).
  • Mandate code examples for public APIs.
  • Check for active voice and terminology.
# _authord/governance/rules.yaml
rules:
- id: require-examples
match: "public-api"
message: "Must include a usage example."
severity: error
- id: terminology-check
banned: ["master", "slave"]
severity: warning

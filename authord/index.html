Authord

Supercharge your VS Code documentation workflow with Authord: a modern, intuitive, and powerful extension designed to streamline your writing process.

Features Usage Writerside Compatibility Known Issues GitHub

Features

  • Documentations Tree View: Easily navigate through your documentation instances defined in authord.json.
  • Topics Tree View: Quickly browse topics for each documentation instance, ensuring effortless organization.
  • Dynamic Updates: Real-time synchronization in the topics directory keeps your documentation current.
  • Open Topics: Instantly open any Markdown file for fast edits.
  • Side-by-Side Preview: Streamlined editing with a live preview, boosting productivity.

Usage

Simply click the Authord icon in the Activity Bar of VS Code to start browsing and managing your documentation.

Writerside Compatibility

Continue your JetBrains Writerside projects seamlessly within Authord. While you can't create new Writerside projects directly here, you can easily pick up where you left off and harness the benefits of both platforms.

Known Issues

Currently, Authord supports a single instance in authord.json. Multi-instance capabilities are planned for future releases.
